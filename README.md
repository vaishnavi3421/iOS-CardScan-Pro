# CardScan Pro

Enterprise-level Business Card Scanner application built using UIKit + SwiftUI + MVVM + Clean Architecture.

## Features
- OCR Card Scanning
- VisionKit Integration
- Offline Storage
- Core Data Support
- Background Sync
- API Integration
- Search & Filters
- Token Authentication

## Architecture
- MVVM
- Clean Architecture
- Repository Pattern

## Tech Stack
- Swift
- UIKit
- SwiftUI
- VisionKit
- Core Data
- URLSession
- Async/Await
